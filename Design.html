Introduction

My website is designed to be semi-serious, with a relaxed atmosphere created by the use of lilac and turquoise colors.

Site map

This layout follows the three-click rule and boosts accessibility by placing all the same items in the footer. Users can access any content from any page.

Design mockups

The expected design for this website was created in a sketch.


I designed my layout with two breakpoints in mind: one for tablet users (min width: 600px) and one for desktop users (min width: 1000px). The only difference between the layouts for these two breakpoints is the font and image sizes.

Menu system

An optimized "do nothing" approach was chosen for the responsive navigation bar because it is the simplest and most effective way to handle responsive navigation bars. It requires no extra code or special handling, and it is easy to implement. This approach is best suited for small sites with few pages and simple navigation, which my portfolio website is an example of. I used a design similar to the university website's, where the logo moves down and the nav bar is above on smaller screens. However, on my website, the nav bar goes below the logo on smaller screens. This makes the website more user-friendly and easily navigable on all devices.
